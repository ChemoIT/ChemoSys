Create `src/components/app/fleet/vehicles/AddVehicleDialog.tsx` — a 'use client' component mirroring the pattern of `src/components/app/fleet/drivers/AddDriverDialog.tsx` but with a two-step MOT flow.

**Props:** `{ open: boolean; onClose: () => void }`

**State:**
- `step`: `'input' | 'previewing' | 'creating'`
- `plate`: string (license plate input)
- `companyId`: string (selected company)
- `motData`: `MotVehicleData | null` (from lookupVehicleFromMot)
- `companies`: `{ id: string; name: string }[]` (from getCompaniesForSelect)
- `isLooking`: from `useTransition` (MOT lookup loading)
- `isCreating`: from `useTransition` (create loading)

**On dialog open (useEffect):** Reset all state. Call `getCompaniesForSelect().then(setCompanies)`.

**Step 1 — Input:**
- License plate input field (text, dir="ltr", font-mono, placeholder "הכנס מספר רישוי")
- Company select dropdown (populated from companies state) — use a native `<select>` with RTL styling (same pattern as other fleet forms) or shadcn Select
- Button "בדוק ב-MOT" — calls `lookupVehicleFromMot(plate)` via `startLookupTransition`
  - On success: set motData, setStep('previewing')
  - On error: `toast.error(result.error)`
  - Disabled if plate is empty or companyId is empty

**Step 2 — Preview:**
- Show MOT data in a read-only card: tozeret_nm (יצרן), degem_nm (דגם), kinuy_mishari (כינוי מסחרי), shnat_yitzur (שנת ייצור), tzeva_rechev (צבע), sug_delek_nm (דלק), baalut (בעלות)
- Show the plate number formatted via `formatLicensePlate` from `@/lib/format`
- Button "פתח כרטיס רכב" — calls handleCreate via `startCreatingTransition`:
  1. `createVehicle(plate, companyId)` — if error, toast.error and return
  2. `syncVehicleFromMot(createResult.vehicleId!, plate)` — fire-and-forget, don't block on failure
  3. `toast.success('כרטיס רכב נפתח')`
  4. `onClose()`
  5. `router.push(\`/app/fleet/vehicle-card/${createResult.vehicleId}\`)`
- Button "חזור" — goes back to step 'input', keeps plate+company values
- If MOT sync fails after create: show toast warning "כרטיס נפתח, סנכרון MOT נכשל" — don't block navigation

**Styling:** Mirror AddDriverDialog — `DialogContent className="max-w-lg" dir="rtl"`, same button styles, same loading spinner pattern with `Loader2`.

**Icons:** Car from lucide-react for dialog title, RefreshCw for MOT lookup button.
