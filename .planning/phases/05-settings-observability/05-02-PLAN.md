**1. `src/app/(admin)/admin/audit-log/page.tsx` (Server Component)**

Follow the URL search params server-side filtering pattern from RESEARCH.md Pattern 2.

- `await verifySession()` as first line
- Await `searchParams` (Next.js 16 Promise pattern): `const params = await searchParams`
- Accepted search params: `entity` (string), `action` (string), `search` (string), `from` (ISO date), `to` (ISO date), `page` (number string)
- Parse page number: `parseInt(params.page ?? '1', 10)`
- PAGE_SIZE = 50
- Calculate range: `from = (page - 1) * PAGE_SIZE`, `to = from + PAGE_SIZE - 1`
- Build Supabase query on `audit_log` table:
  - `.select('id, created_at, action, entity_type, entity_id, user_id, old_data, new_data', { count: 'exact' })`
  - `.order('created_at', { ascending: false })`
  - `.range(from, to)`
  - If `params.entity`: `.eq('entity_type', params.entity)`
  - If `params.action`: `.eq('action', params.action)`
  - If `params.from`: `.gte('created_at', params.from + 'T00:00:00.000Z')`
  - If `params.to`: `.lte('created_at', params.to + 'T23:59:59.999Z')`
  - If `params.search`: `.or(\`entity_type.ilike.%${params.search}%,entity_id::text.ilike.%${params.search}%\`)`
- Resolve user display names using the two-step pattern:
  - Collect distinct `user_id` values from the fetched rows
  - Query `public.users` with `.select('auth_user_id, full_name, email').in('auth_user_id', distinctUserIds)`
  - Build a `Map<string, string>` mapping user_id -> full_name (fallback to email, then truncated UUID)
  - Merge userName into each row before passing to client
- Fetch distinct entity types and action types for filter dropdowns:
  - Two separate queries on `audit_log`: `SELECT DISTINCT entity_type` and `SELECT DISTINCT action` (use `.select('entity_type')` with limit 1000, then deduplicate in JS with `Set`)
- Render: page header with "יומן פעולות" title + `<RefreshButton />`, then `<AuditLogTable>` with all data and filter state
- Pass to AuditLogTable: `rows` (with userName merged), `totalCount`, `pageSize`, `currentPage`, `entityTypes` (string[]), `actionTypes` (string[]), `currentFilters` (the params object)
