**2. `parseMoedAliya(raw)` helper:**
- Input: `"YYYY-M"` or `"YYYY-MM"` format from MOT API
- Output: `"YYYY-MM-01"` ISO date string for PostgreSQL DATE column
- Return null if raw is falsy or unparseable

**3. `syncVehicleFromMot(vehicleId: string, licensePlate: string)`:**
- `verifySession()` guard
- Strip non-digits from licensePlate, convert to Number — **CRITICAL: MOT API requires numeric, not string**
- If plateNumber is NaN or 0, return `{ success: false, error: 'מספר רישוי לא תקין' }`
- Build URL: `https://data.gov.il/api/3/action/datastore_search`
  - `resource_id`: `053cea08-09bc-40ec-8f7a-156f0677aff3`
  - `filters`: `JSON.stringify({ mispar_rechev: plateNumber })` — **Number, NOT string**
  - `limit`: `1`
  - `records_format`: `objects`
  - `include_total`: `true`
- Fetch with `{ headers: { 'Accept': 'application/json' }, next: { revalidate: 0 } }` — **never cache MOT data**
- If !res.ok → return network error
- If total === 0 → return `{ success: false, error: 'הרכב לא נמצא ברשומות משרד הרישוי' }`
- Extract first record `v = json.result.records[0]`
- UPDATE `vehicles` table (Supabase client):
  - Set MOT fields: tozeret_nm, degem_nm, kinuy_mishari, shnat_yitzur, tzeva_rechev, sug_delek_nm, misgeret, degem_manoa, ramat_gimur, kvutzat_zihum (String), baalut, moed_aliya_lakvish (via parseMoedAliya), mot_last_sync_at (now)
  - `.eq('id', vehicleId)`
- If MOT record has `mivchan_acharon_dt` AND `tokef_dt` — INSERT into `vehicle_tests`:
  - `vehicle_id`, `test_date` = mivchan_acharon_dt, `expiry_date` = tokef_dt, `passed` = true, `alert_enabled` = true
  - `created_by` + `updated_by` = user's authUserId
  - **Simple INSERT, not upsert** — let test history accumulate. If duplicate test_date exists for this vehicle, the INSERT will just add another row (no unique constraint on vehicle_id+test_date).
- `revalidatePath('/app/fleet/vehicle-card/' + vehicleId)`
- Return `{ success: true, data: v }`

**4. `testMotApiConnection()`:**
- `verifySession()` guard
- Test with a known plate number (e.g., `6242255` — a public bus, always in registry)
- If API returns success with total > 0 → `{ ok: true, message: 'API משרד הרישוי נגיש — מצא רכב בדיקה' }`
- If API returns success but total 0 → `{ ok: true, message: 'API נגיש אך לא מצא רכב בדיקה' }`
- If fetch fails → `{ ok: false, message: 'שגיאת רשת: ...' }`
- Return type: `TestResult` (import from settings.ts)

Important:
- Use `import { verifySession } from '@/lib/dal'`
- Use `import { createClient } from '@/lib/supabase/server'`
- Use `import type { TestResult } from '@/actions/settings'`
- Do NOT import or use writeAuditLog for MOT sync — it's a data sync, not a user mutation
- The `kvutzat_zihum` field must be `String(v.kvutzat_zihum ?? '')` (convert number to string for DB TEXT column)
