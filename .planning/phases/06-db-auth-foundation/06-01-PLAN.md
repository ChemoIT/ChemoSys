After `const session = await verifySession()`, add:

```typescript
// Guard: block non-admin users from the admin shell.
// is_admin flag lives in public.users table (added in migration 00012).
// maybeSingle() returns null if no row exists ÔÇö that means bootstrap admin (Sharon
// before first user creation) ÔåÆ allow access.
// userRow.is_admin === false ÔåÆ regular ChemoSys user ÔåÆ redirect to app login.
const supabase = await createClient()
const { data: userRow } = await supabase
  .from('users')
  .select('is_admin')
  .eq('auth_user_id', session.userId)
  .is('deleted_at', null)
  .maybeSingle()

if (userRow !== null && !userRow.is_admin) {
  redirect('/chemosys')
}
```

**Critical:** Use `.maybeSingle()` NOT `.single()`. The bootstrap admin (Sharon's auth user before public.users row was created) must still access admin ÔÇö `.single()` would throw on 0 rows.

Do NOT move `verifySession()` ÔÇö it must remain the first call. The is_admin guard comes after it because it needs `session.userId`.
